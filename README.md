# TensorFlow 2.0 使用流程

## 项目简介

- 本项目的目的是梳理自己在学习 TensorFlow 2.0 时的知识点以及编程思路，也为了方便他人在学习 TensorFlow 2.0 时能够多一份参考资料，能快速入门 TensorFlow 2.0 。
  
## 项目目录结构

```C
doc\        # 存放教程
pic\        # 存放教程中所使用的图片
src\        # 项目源码
test_image\ # 测试图片
LICENSE     # 项目许可证
README.md   # 项目简介文件
.gitignore  # 上传到 GitHub 时，忽略文件
```
